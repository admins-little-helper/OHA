# Get-OHASession

## SYNOPSIS

Returns the session variable hashtable.

## SYNTAX

```PowerShell
Get-OHASession
```

## DESCRIPTION

The 'Get-OHASession' function returns the session variable hashtable.

## EXAMPLES

### EXAMPLE 1

```PowerShell
Get-OHASession
```

Returns the session variable hashtable.
